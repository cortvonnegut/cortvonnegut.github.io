Journal Entries

11/20/2019

This is my first journal entry on my unregistered webpage... I feel like Creed when they tell him he has a blog, but it's not actually live. That's probably for the best. Way less pressure for any of this to actually be good. I've been tossing around the idea of learning to code for a few years now, but I always found it really overwhelming. I never thought I would be able to learn it, but it turns out it's a lot easier than I expected. Especially if you have a nice text editor like Sublime that helps track your changes.

I wasn't sure what I was hoping to get out of this, but it turns out it's just kind of fun to be able to create things. This is all hopefully building to designing my own website and beginning to put out the kind of content that I want to see. It's really nice to have the kind of control that you get when you're building it from the ground up. I just started doing this yesterday, and already I can do enough to build a basic webpage, and you know what? That's something! As I'm learning more, this will hopefully get more complex.
